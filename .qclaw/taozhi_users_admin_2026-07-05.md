# 桃之 - 用户管理系统补全 (2026-07-05 11:50)

## 新增

### admin/users.php（用户管理后台）
- 自动建表（users 表）
- 用户列表：ID、头像、昵称、手机号、OpenID、余额、积分、状态、注册时间
- 搜索：按昵称/手机号/OpenID 模糊搜索
- 分页：20人/页
- 操作：调整余额（设为/增加/减少）、调整积分、启用/禁用用户
- 弹窗：余额弹窗（支持三种模式）、积分弹窗
- 遮罩点击关闭弹窗

### sidebar.php 更新
- 新增「👥 用户管理」导航项

### admin/index.php 更新
- 统计卡片改为4列（新增注册用户数）
- 快捷操作新增「👥 用户管理」
- 系统状态新增用户数量

### 数据流
```
微信小程序「我的」→ 点登录 → wx.login() + 后端 user.php → users 表创建用户
→ 用户出现在 admin/users.php 列表
→ 管理员可调整余额/积分/禁用用户
→ 地址关联 user_id
```
